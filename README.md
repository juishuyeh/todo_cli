一個簡潔優雅的終端機待辦事項管理工具，使用 Go 和 SQLite 構建。

## ✨ 功能特點

- 🚀 快速啟動，無需配置
- 💾 本地 SQLite 儲存
- 🎨 美觀的 TUI 介面
- ⌨️ 鍵盤快捷鍵操作
- 📦 單一二進制檔案，易於分發
